## J&M Rentals Website
This project is a website written for J&M Rentals LLC located in Larose, Louisiana. J&M Rentals is a company which rents out bayou-side apartment houses. This website was made to serve as a central location to showcase to new, current, and old customers about J&M Rentals. 

### Plan
The plan is for this website to serve as a managerial hub for future expansion of J&M Rentals. Plans include:
- Booking of new aprtments for new guests
- Guest portal to pay rent
- Contracts/leases/agreements all signed online
